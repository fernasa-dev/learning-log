## Comandos Nuevos Aprendidos
| Comando | Qué Hace |
|---|---|
| `git init` | Inicia un repositorio Git en una carpeta |
| `git add .` | Prepara todos los archivos para el commit |
| `git commit -m ""` | Guarda los cambios con un mensaje |
| `git push -u origin main` | Sube el código a GitHub |
| `git config --global` | Configura opciones globales de Git |
| `code .` | Abre VS Code en la carpeta actual |
| `type()` | Para inspeccionar variables |
| `f""` | Combinar texto y variables|
| `input()` |Solicitar datos del usuario|

### Errores que resolví
- **Error:** `git add .git commit` pegados en una línea
  - **Fix:** Ejecutar cada comando por separado con Enter

- **Error:** GH007 email privacy bloqueó el push
  - **Fix:** Desactivar "Block command line pushes" en GitHub Settings → Emails

- **Error:** `sueldo_actual` aparecía como `int` en lugar de `float`
  - **Fix:** Escribir `20500.0` o `20500.50` para que Python lo interprete como float
